# r4subprofile: Regulatory Submission Profiles for Clinical Submission Readiness

Defines submission profiles per regulatory authority with
authority-specific pillar weights, decision thresholds, indicator
requirements, and risk configuration. Supports FDA, EMA, PMDA, Health
Canada, TGA, and MHRA. Integrates with 'r4subscore' and 'r4subrisk'
configuration systems.
